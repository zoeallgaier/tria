# Tria

**Social Media is so back.**

Tria is a quiet social app for small circles of friends. It's built to be
low-tech, non-addictive, and anti-bloat: no infinite feed, no ads, no metrics to
chase. Posts have room to breathe, so context and real writing are the point.

Live at **[triaonline.com](https://triaonline.com)**.

## What's inside

- **My Circle** — a calm feed of your friends, newest first, with notes, finds
  (links worth sharing), photos, and activities.
- **Activities** — plans with a place and a time, RSVPs, and an add-to-calendar
  button.
- **Friends** — mutual, request-based. Small by design.
- **Comments and private likes** — a like is a quiet nod, not a public tally.
- **Updates** — a gentle, pull-based digest of what happened on your posts.
- Installable as a home-screen app, with optional push notifications.

## Built with

Plain HTML, CSS, and JavaScript with a tiny hash router — no framework and no
build step — on a [Supabase](https://supabase.com) backend (auth, database, and
storage). Hosted on GitHub Pages.
